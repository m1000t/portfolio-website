Constants/Variables

What they are?

Variables are values that programmers can modify at any time. This is significant because it enables dynamic programming that can instantly react to human input.
There are four different sorts of variables: floats (decimal numbers), ints (whole numbers), Boolean (which can only be true or false), and strings (a series of characters).
However, a constant is a piece of data whose value is fixed throughout the duration of the program. As a result, the value is constant, just as its name suggests.
